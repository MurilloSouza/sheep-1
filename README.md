# SHEEP - Script to find md5 and sha1

####Installing:

```
     sudo gem install httparty
     git clone https://github.com/InploitSec/sheep/
     cd sheep
```

####Commands:

```
      COMMAND          FUNCTION
      --online         Decrypt online.
      --offline        Decrypt offline.
      -m --md5         Setting Hash type md5.
      -s --sha1        Setting Hash type sha1
      Examples:
      ruby sheep.rb --online -m 23ec24c5ca59000543cee1dfded0cbea
      ruby sheep.rb --online -s 6525f2e663e8d223c2f9041fefdf2dba6ca70d63
      ruby sheep.rb --offline --md5 23ec24c5ca59000543cee1dfded0cbea
      ruby sheep.rb --offline --sha1 6525f2e663e8d223c2f9041fefdf2dba6ca70d63
```
